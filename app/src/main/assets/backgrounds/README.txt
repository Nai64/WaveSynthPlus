BACKGROUND IMAGES
=================

This folder contains all background images for WaveSynth.

HOW TO ADD NEW BACKGROUNDS:
---------------------------
1. Drop any PNG or JPG image file into this folder
2. Name it whatever you want (e.g., "Galaxy.png", "Ocean.jpg")
3. The filename (without extension) will appear in the background chooser
4. Rebuild the app - that's it!

TIPS:
-----
- Recommended size: 1920x1080 or higher for best quality
- PNG files support transparency (if you want that)
- JPG files are smaller in size
- File names are sorted alphabetically in the menu
- Use descriptive names - they'll be shown to users
