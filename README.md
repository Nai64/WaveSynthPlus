WaveSynthPlus is a modernized fork of WaveSynth (formerly EasySynth), featuring a completely redesigned Material Design UI and enhanced functionality. This Android synthesizer manipulates harmonics over time using additive synthesis techniques, with support for filtering simulation and FM synthesis.

## ğŸš€ Getting Started

### Prerequisites
- Android device or emulator with API 28+ (Android 9.0 Pie)
- Android Studio for development

### Building from Source

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nai64/WaveSynthPlus.git
   cd WaveSynthPlus
   ```

2. **Open in Android Studio**
   - Launch Android Studio
   - Select "Open an Existing Project"
   - Navigate to the cloned directory

3. **Build the project**
   ```bash
   ./gradlew assembleDebug
   ```

4. **Install on device**
   ```bash
   ./gradlew installDebug
   ```
